Dockerfile

Dockerfile分为四个部分: 基础镜像信息、维护者信息、镜像操作指令和容器启动指令。
即FROM、MAINTAINER、RUN、CMD四个部分

指令说明

nginx-dockerfile示例

vim Dockerfile

需要先下载nginx-1.14.0.tar.gz在Dockerfile同级目录下，然后执行如下命令
docker build -t nginx_image ./Dockerfile
