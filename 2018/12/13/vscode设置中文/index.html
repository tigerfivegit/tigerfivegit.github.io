vscode安装

打开https://code.visualstudio.com选择版本下载

vscode设置中文

首先下载插件：Chinese (Simplified) Language Pack for Visual Studio Code
https://img2018.cnblogs.com/blog/954415/201809/954415-20180915154430076-118026477.png

在Chinese (Simplified) Language Pack for Visual Studio Code界面中点击Reload（重新加载）

其次使用快捷键在Ctrl+Shift+p 打开命令面板 输入“config” 进入locale.json（配置显示语言）
将”locale”设置为”locale”:”zh-cn”
https://img2018.cnblogs.com/blog/954415/201809/954415-20180915155103896-551631575.png

保存 关闭vscode编辑器 等待几秒重新配置好后 重新启动便是中文界面
