13、元组.md

Posted on 2022-04-06 In python

一、简介

  1. 在 Python 中，元组是一种不可变序列，它使用圆括号来表示。
  2. 不可变的tuple有什么意义？因为tuple不可变，所以代码更安全。
  3. tuple也是 python 中一个标准的序列类型。
  4. Python 的元组与列表类似，不同之处在于元组的元素不能修改。
  5. 元组使用小括号，列表使用方括号。
  6. 元组创建很简单，只需要在括号中添加元素，并使用逗号隔开即可
  7. 如果元组的元素只有一个可以省略不写

六、常见元组常量和运算总结

运算 解释
() 空元组
tup = (0,) 单个元素的元组（非表达式）
tup = (0,’str’,1.2,3) 四个元素的元组，混合类型
tup = 0,’str’,1.2,3 四个元素元组，与前列相同

tup = tuple(‘hello’) 一个可迭代对象的项转换为元组
tup[i], tup[i], tup[start:end],len(tup) 索引，索引的索引，分片，长度
tup+tup2, tup2 * 3 合并，重复(元素项重复)
for x in tup : print(x) 迭代
‘元素’ in tup 成员关系
[i ** 2 for i in tup] 列表解析
tup.index(‘元素’) 搜索
tup.count(元素) 计数
