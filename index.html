1.Log4j组件

####1.1. Loggers(记录器)
Loggers组件在此系统中被分为五个级别：

五个级别的顺序,分别用来指定这条日志信息的重要程度：

DEBUG < INFO < WARN < ERROR < FATAL

Log4j规则：只输出级别不低于设定级别的日志信息，假如Loggers级别设定为INFO，则INFO、WARN、ERROR和FATAL级别的日志信息都会输出，而级别比INFO低的DEBUG则不会输出。
